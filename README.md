## Description
This application allows users to monitor their burger consumption. The user can first add burgers to their list of burgers to eat. Then, once they are finished eating the burger, they can click the button confirming that they have finished eating the burger and the burger will be added to the "devoured" list.
## Installation Instructions
The user should input the name of the burger into the input field and then click the 'submit' button. Once they have finished eating the burger, the user should click the 'devoured' button to indicate that they have finished eating that particular burger.
